# Norse-Votes :mega:
* Mobile first, real time voting application developed to engage the Luther College Community.

## Built with
* Node.js
* Express.js
* MongoDB
* Pusher
* Materializecss

## Features
* Real time event propagation with Pusher
* AuthO
